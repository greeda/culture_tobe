# 스크립트 관련 가이드
# 기본은 전체가 열리는 버전 입니다.
# .accordion-list 에 ui-accordion-toggle 클래스 붙이면 토글 형태로 변경됩니다.

# 스크립트 관련 가이드

# option 속성 .accordion-list(data-group="group2" data-motion-type="static" data-body-scroll="false")
    group : "string"
        ex ) 스트링이 같은것끼리 만 닫힘, 다른건 안닫힘 data-group 속성값 없으면 각각 열리고 닫는 토글기능
    motion-type : "string"
        ex ) 기본 slide , option : static
    body-scroll : "boolean"
        ex ) 기본 true 해당 컨텐츠로 포커스 이동 시킵니다 . false하면 포커스 안됨.
    toggle-group : "string"
        ex ) 모두 열기 ,닫기 기능있을때 data-toggle-group 속성을 버튼과 .accordion-list 에 똑같이 넣어줘야함
