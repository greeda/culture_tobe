# .form-validation-container 영역에 추가 클래스로 컨텐츠간 여백을 조정 합니다.

# 벨리데이션 체크
# class="form-validation-container pass" - 성공 스타일
# class="form-validation-container error" - 에러 스타일
# class="form-validation-container disabled" - 비활성 스타일
# class="form-validation-container readonly" - 비활성 스타일
# 위 클래스를 상속 받아서 하위 영역을 컨트롤 합니다.

# class="input-txt" - 하단 라인형 스타일
# class="input-txt line-all" - 전체 라인형 스타일
# class="input-txt btn-type01" - 버튼 추가형 스타일

# input text은 label 태그를 필수로 연결해야 합니다. label를 사용할 수 없을때는 title="" 태그를 사용 합니다.
# input의 id 값과 label의 for 값을 동일하게 연결해줘야 합니다.
