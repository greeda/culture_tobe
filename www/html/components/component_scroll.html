# class="ui-scrollbar" 클래스가 있어야 작동 합니다.
# class="ui-scrollbar scrollbar-inner" 클래스는 스크롤바가 텍스트의 안쪽에 위치합니다.
# class="ui-scrollbar scrollbar-outer" 클래스는 스크롤바가 텍스트의 바깥쪽에 위치합니다.

# scrollbar-inner, scrollbar-outer 영역에 max-height 속성으로 스크롤 범위를 지정 합니다.
# scrollbar-inner, scrollbar-outer 영역은 jquery.scrollbar 라이브러리에서 사용하는 클래스 입니다.
# ui-scrollbar 클래스는 common.js 에서 컨트롤 합니다.

# scrollbar-inner, scrollbar-outer 영역에는 padding 속성을 넣을 수 없습니다.
# scrollbar-inner, scrollbar-outer 자식 엘리먼트에서 padding 속성을 조정 합니다.
