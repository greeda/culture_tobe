# 버튼의 스타일은 .blue, .white 등의 추가 클래스를 만들어서 활용 합니다.

# 버튼의 텍스트 영역을 span 마크업으로 확장해서 사용하면 마우스 hover 애니메이션 효과에 대하여 쉽게 대응 할 수 있습니다.

# 아이콘 영역은 빈 태그이며 보이스 오버나 스크린리더에 읽히지 않도록 aria-hidden="true" 처리 해줍니다.

# 링크 버튼과 기능 버튼을 구분해서 사용 합니다.
# <a></a> 태그는 링크 버튼에 사용합니다.
# <button type="button"></button> 태그는 링크 이동이 아닌 기능 버튼에 사용합니다. (type속성을 명시해 주세요.)
# <a role="button"></a> 태그를 기능 버튼으로 사용해야 하는 경우는 aria 태그의 role="button" 속성을 넣어 줍니다.
